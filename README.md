# I Like E ❤️


### [I like E 브런치스토리로 이동](https://i-like-e.netlify.app/)

<br/>
<hr/>

## 📢 프로젝트 소개

**🧠'어디까지 만들 수 있을까..?'** <br/>
"기능 하나라도 우리 손으로 직접 완성해보자" 라는 마음으로
팀원들과 함께 **브런치스토리**를 구현했습니다.

<br/>

## 🌱 주요 기능

### 회원 관련 기능 <br/>

- 이메일 로그인

### 공통 컴포넌트 <br/>

- 헤더 컴포넌트 (로그인/비로그인 상태)
- 푸터 컴포넌트
- 네비 컴포넌트

### 📚 내 서랍 페이지 <br/>

- 비로그인 상태일 때 로그인 페이지로 이동
- 구독 작가 목록
- 최근 본 글 목록
- 관심 글 목록
- 나의 브런치 목록

### 기타 <br/>

- API 통신 & 에러 처리

<br/>
